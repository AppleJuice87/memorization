# blank-2-memory
# 📝 Tistory Study Skin (hELLO 기반 암기용 커스텀 스킨)

티스토리 블로그를 단순한 기록 공간을 넘어, **강력한 인터랙티브 학습 플랫폼**으로 만들어주는 커스텀 스킨 코드입니다. 본 프로젝트는 [hELLO 스킨](https://github.com/pronist/hello)을 기반으로 동작하며, 방대한 암기량이 요구되는 시험을 대비하기 위한 마스킹 및 빈칸 채우기 학습 기능을 제공합니다.

## 🛠 주요 기능 (Features)

본 스킨은 우측의 플로팅 패널을 통해 다양한 학습 모드를 실시간으로 전환할 수 있습니다.

### 📚 학습 모드
* **정답 보기:** 마스킹된 정답 텍스트를 초록색 굵은 글씨로 강조하여 빠르게 회독할 수 있습니다.
* **마스킹 모드:** 정답이 가려집니다. 마우스를 올리면(Hover) 정답을 확인할 수 있습니다.
* **빈칸 채우기:** 정답 영역이 입력 가능한 텍스트박스로 변환됩니다.
  * 정답 입력 후 `Enter` 키를 누르면 즉시 채점됩니다 (정답: 초록색 / 오답: 빨간색).
  * 채점 후 자동으로 다음 빈칸으로 커서가 부드럽게 이동합니다.
* **첫글자 힌트:** 빈칸 채우기와 동일하나, 텍스트박스에 정답의 첫 글자가 힌트로 제공됩니다.
* **타이핑 연습:** 텍스트박스 배경에 연한 회색으로 정답 전체가 표시되어, 글씨를 따라 치며(Tracing) 암기할 수 있습니다.
* **오답만 남기기 (Spaced Repetition):** 학습 효율을 위해 붉은색으로 체크된 오답과 미입력 빈칸만 남기고, 맞춘 정답은 일반 텍스트로 변환하여 아는 것을 지워나가는 학습이 가능합니다.

### ✏️ 커스텀 편집 (관리자 모드)
웹 에디터 상에서 직관적으로 학습용 빈칸을 생성하고 저장할 수 있습니다.
* `드래그`: 텍스트를 드래그하여 즉시 마스킹 영역(빈칸)으로 지정
* `우클릭`: 지정된 마스킹 영역 해제
* `Ctrl/Cmd + Z`: 실행 취소 (Undo) 기능 지원 (최대 20개 히스토리)
* `ESC`: 편집 모드 빠른 종료
* `저장 (클립보드 복사)`: 마스킹 작업이 완료된 HTML을 클립보드에 복사하여 티스토리 게시물에 손쉽게 반영

### 🎨 부가 기능 (UI/UX)
* **동적 텍스트박스 너비:** 한글과 영문의 폭을 계산하여 정답 길이에 딱 맞는(최소 7글자 보장) 자연스러운 텍스트박스를 생성합니다.
* **옵시디언(Obsidian) 스타일 리스트:** 중첩 리스트(ul, ol) 사용 시 세로 가이드라인이 생성되며, 마우스 오버 시 하이라이트 되어 글의 구조를 쉽게 파악할 수 있습니다.

## 🚀 적용 방법 (Getting Started)

본 코드는 티스토리 블로그에 **hELLO 스킨**이 선행 적용되어 있어야 정상적으로 동작합니다.

1. 티스토리 관리자 페이지 ➔ [꾸미기] ➔ [스킨 편집] ➔ **[html 편집]** 으로 이동합니다.
2. **`CSS` 적용:** 프로젝트의 `CSS.css` 파일 내용을 티스토리 CSS 탭의 가장 밑에 복사하여 붙여넣습니다.
3. **`HTML / JS` 적용:** 프로젝트의 `HTML.html` 파일 내용을 티스토리 HTML 탭의 `</body>` 태그 **바로 위**에 복사하여 붙여넣습니다.
4. 우측 상단의 [적용] 버튼을 누르면 설정이 완료됩니다.

<br>

## 💡 게시글 작성 가이드 (Usage)

1. 티스토리 에디터(마크다운 등)를 이용해 평소처럼 요약 노트를 작성하고 비공개/공개 발행합니다.
2. 발행된 게시물에 접속하여 우측 플로팅 메뉴의 **[커스텀 편집]** 버튼을 클릭합니다.
3. 문제로 만들고 싶은 키워드나 문장을 **드래그**하여 빈칸으로 만듭니다.
4. 작업이 끝나면 하단 패널의 **[저장 (클립보드 복사)]** 버튼을 클릭합니다.
5. 해당 게시물의 수정 화면으로 들어가 기본 에디터 모드를 **[HTML]**로 변경합니다.
6. 기존 내용을 모두 지우고 복사된 내용을 붙여넣기(`Ctrl/Cmd + V`) 한 뒤 다시 발행하면 완성됩니다!
